# RNN_for_chatbot
Application d'un réseau de neurones séquentiel pour la création d'un chatbot


Word embedding:
Utilisation du pré-trained word embedding:

Lorsque nous disposons de si peu de données pour apprendre une intégration spécifique à une tache appropriée du vocabulaire, au lieu d'apprendre les intégrations de mots conjointement avec le problème, nous pouvons charger des word embedding vectors à partir d'un espace d'intégration précalculé dont on sait qu'il est hautement structuré et présente des propriétés utiles, qui capturent les aspects génériques de la structure du langage.


Transfer learning: 
Consiste à utiliser des modèles pré-entrainés sur de grandes quantités de données textuelles, pour améliorer les performances pendant le procecessus de formation.
